## Installation
- Install node js ( <a href="https://nodejs.org/en/download/">Download Here</a> )
- clone this repo ( git clone https://github.com/Idxxlism/stumble-project )
- Goto this directory ( cd stumble-project )
- Install It ( npm install )
- Run ( npm start )
